팬 혜택 시스템

소장하면 받는 실물 혜택

투기가 아닌 진짜 팬덤을 위한 보상입니다. NFT를 소장하면 아티스트가 직접 준비한 혜택이 자동으로 지급됩니다.

🎟️

공연 티켓 응모권

NFT 소장자에게 아티스트 공연 티켓 우선 응모 자격 자동 부여. NFT 보유 수량에 따라 당첨 확률 증가.

선착순 응모
💿

실물 음반 할인권

NFT 소장 시 해당 아티스트의 실물 음반·굿즈 구매 시 최대 30% 할인쿠폰 자동 발급.

즉시 발급
🔐

홀더 전용 채팅방

NFT 소장자만 입장 가능한 아티스트 전용 채팅방 자동 연동. 미공개 곡 미리 듣기, 비하인드 공유 등.

자동 연동
🎁

아티스트 굿즈 교환권

포토카드, 클리어파일, 포스터 등 아티스트가 직접 제작한 굿즈와 교환 가능한 쿠폰 지급.

실물 교환
🌟

팬미팅 우선 초대

아티스트가 진행하는 온/오프라인 팬미팅에 일반 팬보다 우선 초대 혜택. 소장 NFT 수에 따라 등급 차등.

등급 혜택
📨

채팅창 뮤직 카드

소장한 NFT를 텔레그램 채팅방에 공유하면 즉시 재생 가능한 뮤직 카드로 표시. 바이럴 자동 확산.

소셜 공유
이용 방법

아티스트라면 이렇게

01

텔레그램 봇 실행

별도 앱 설치 없이 텔레그램 봇 링크 클릭 한 번으로 즉시 시작.

02

음원 업로드 & 저작권 검증

MP3/WAV 업로드 시 오디오 핑거프린팅으로 자동 저작권 검증. 본인 창작곡만 등록 가능.

03

소장형 NFT 발행

곡명·설명 입력 후 'Mint' 클릭 → TON 블록체인에 영구 기록. 소장용 1:n 컬렉터블로 발행.

04

팬 혜택 설정

소장 팬에게 줄 혜택(공연 티켓 응모권, 굿즈 할인, 팬미팅 초대 등)을 직접 설정.

05

채널 공유 & 팬과 소통

텔레그램 채널에 뮤직 카드로 공유. 소장자 전용 채팅방에서 팬과 직접 소통.
